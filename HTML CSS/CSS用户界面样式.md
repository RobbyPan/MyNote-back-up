# CSS用户界面样式

###  标样式cursor

li{cursor:pointer;}

设置光标的样式

| 属性值      | 描述 |
| ----------- | ---- |
| default     | 默认 |
| pointer     | 小手 |
| move        | 移动 |
| text        | 文本 |
| not-allowed | 禁止 |

### 轮廓线outline

input{outline:none;} 或者outline:0；去掉默认蓝色边框

### 防止拖拽文本域resize

textarea{resize:none;}

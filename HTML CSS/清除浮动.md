# 清除浮动

### 为什么清除浮动

1.父级没高度

2.父盒子浮动

3.影响下面布局

### 清除浮动

| 清除浮动的方式       | 优点           | 缺点                        |
| -------------------- | -------------- | --------------------------- |
| 额外标签法（隔墙法） | 通俗易懂       | 添加无意义标签，结构化较差  |
| 父级overflow:hidden; | 书写简单       | 溢出隐藏                    |
| 父级after伪元素      | 结构语义化正确 | IE6-7不支持:after，兼容问题 |
| 父级双伪元素         | 结构语义化正确 | IE6-7不支持:after，兼容问题 |

# CSS三大特征

### 层叠性、继承性、优先级

| 选择器               | 选择器权重 |
| -------------------- | ---------- |
| 继承或者*            | 0,0,0,0    |
| 元素选择器           | 0,0,0,1    |
| 类选择器，伪类选择器 | 0,0,1,0    |
| ID选择器             | 0,1,0,0    |
| 行内样式style=""     | 1,0,0,0    |
| !important           | 无穷大     |
